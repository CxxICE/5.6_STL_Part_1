# Задача 1. Частота символов

### Описание
Нужно посчитать, сколько раз каждый символ встречается в заданном тексте, и отсортировать по убыванию частоты.

---

### Пример правильной работы программы
```
[IN]: Hello world!!
[OUT]: 
l: 3
o: 2
!: 2
 : 1
H: 1
e: 1
d: 1
w: 1
r: 1
```
